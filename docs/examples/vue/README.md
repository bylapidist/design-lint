# Vue Example

Shows how to lint `.vue` files with design-lint.

## Steps

1. Install dependencies:
   ```bash
   npm install --save-dev @lapidist/design-lint
   ```
2. Lint Vue files:
   ```bash
   npx design-lint "src/**/*.vue"
   ```
